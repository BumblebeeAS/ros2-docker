# Environments

The `.isaac_ros_common-config` file within each environment folder is the main configuration file.

## Argument Descriptions

| Key                         | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             | Example                                                                                            |
| --------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------- |
| `BUILD_IMAGE_FLAG`          | REQUIRED. Whether to build an image.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    | `1`                                                                                                |
| `CONFIG_IMAGE_KEY`          | REQUIRED if `BUILD_IMAGE_FLAG=1`, disregarded otherwise. A list of image keys indicating which Dockerfiles are to be built. Inputted as a concatenation of keys separated by `.`. Each key must match the suffix of a Dockerfile's name. Put keys for Dockerfiles that build dependencies for later Dockerfiles first. For example, in `base.spinnaker`, `base` would be built before `spinnaker`. For Dockerfiles with no cross-dependencies, put Dockerfiles with long build times first. This is because any change to a Dockerfile will trigger builds for Dockerfiles appearing later even if no changes to these later Dockerfiles are made. For example, a change in `base` would trigger a build for both `base` and `spinnaker` when the keys are written as `base.spinnaker`. | `base.spinnaker`                                                                                   |
| `CONFIG_DOCKER_SEARCH_DIRS` | REQUIRED if `BUILD_IMAGE_FLAG=1`, disregarded otherwise. A list of directories where the scripts searches for the Dockerfiles indicated by `DOCKER_BUILD_IMAGE_KEYS`. Take note to enclose it with `()` and ensure that there are no spaces.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            | `()`                                                                                               |
| `BUILT_IMAGE`               | REQUIRED. Name of the Docker image. If `BUILD_IMAGE_FLAG=1`, this will be the name of the resultant image built. If `BUILD_IMAGE_FLAG=0`, the `run_dev.sh` script will search the disk for the image name and an error will be thrown if the image is not found.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        | `ros-humble-desktop-full`                                                                          |
| `MOUNT_DIRS`                | REQUIRED. List of directories to mount (changes inside the container will be reflected outside and vice versa).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         | `("$HOME/workspaces/ros2_ws:/workspaces/ros2_ws" "$HOME/.cache/ccache:/home/admin/.cache/ccache")` |
| `WORKDIR`                   | OPTIONAL, defaults to `/`. Working directory inside a container on start.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               | `/workspaces/ros2_ws`                                                                              |
| `DOCKER_CONTEXT_DIR`        | REQUIRED. See https://docs.docker.com/build/concepts/context/.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          | `$HOME/workspaces/ros2-docker/`                                                                    |
